Wireless Drawing Pad for Nokia 5510
What it does: Draw on your smartphone screen and watch your doodles appear instantly on a retro Nokia 5510 display connected to a Wemos Mini Pro.

How it works:

Wemos Mini Pro creates a WiFi web server

Phone connects to the board's IP via browser

Drawing canvas loads on phone

Strokes appear simultaneously on phone AND Nokia display

Features:

Real-time drawing sync

Adjustable pen size (1-3px)

Clear button

Pixel counter

Auto-clear after inactivity (30s)

Retro 84x48 LCD look

Hardware:

Wemos Mini Pro (ESP8266)

Nokia 5510 display (SPI)

Pins: D1(SCLK), D2(DIN), D5(DC), D6(CS), D7(RST)

Perfect for: Pixel art, messages, retro projects, or just fun drawing from your phone!
